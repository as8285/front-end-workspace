# CSS (Cascading Style Sheet)

- 웹 페이지의 표현(디자인)을 나타내는 언어
- 특정 요소를 선택해서 원하는 " 스타일"과 "기능"을 적용시킬수 있음

```CSS
선택자(누구한테){
    속성(어떤효과):값(얼만큼);
}
```
